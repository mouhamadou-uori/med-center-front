## 🚀 Démarrage Rapide

### Prérequis
- Java 17+
- Node.js 18+
- MySQL 8.0+
- Docker (pour Orthanc)

### Commandes essentielles

**Backend :**
```bash
mvn spring-boot:run
# Accessible sur http://localhost:9000
```

**Frontend :**
```bash
pnpm dev
# Accessible sur http://localhost:3000
```
